## 🚀 Déployer sur GitHub Pages

1. **Créer un repo** sur GitHub (ex: `tribup-brand-board`)
2. **Uploader** les fichiers `index.html`, `style.css`, `README.md`
3. Aller dans **Settings → Pages**
4. Source : `main` branch, dossier `/ (root)`
5. Cliquer **Save** — la page est live en ~60 secondes
